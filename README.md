# Small Business Network Design

## Project Overview

Designed and implemented a secure Cisco-based small business network for HSK Company supporting Management, Operations, and Sales departments.

## Technologies Used

- Cisco Packet Tracer
- Cisco IOS
- VLANs
- Trunking
- Inter-VLAN Routing
- DHCP
- PAT (Port Address Translation)
- SSH
- TFTP Backup Server

## Network Features

- VLAN segmentation
- Router-on-a-Stick configuration
- DHCP automation
- PAT for Internet connectivity
- Secure SSH remote administration
- Configuration backup and restoration using TFTP
